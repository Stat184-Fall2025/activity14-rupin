Activity 14 — Reproducible Workflow Documentation

This repository contains all analysis files, datasets, and version-control work completed for Activity 14.
The purpose of this repo is to demonstrate a clean and organized GitHub workflow, including branches, issues, commit messages, and reproducible project structure using R, Quarto, and Git.

activity14-report.qmd — Main Quarto document containing the full analysis
activity14-report.pdf — Rendered output generated from the QMD
plan.md — Project planning document and workflow outline
README.md — Overview of repository structure and purpose
